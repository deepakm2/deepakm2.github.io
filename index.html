What IS on the Menu!

A culinary trip down the memory lane

About the Visualization

This project uses the hybrid approach of an Interactive Slide-Show to narrate the visual story of NYPL Menus dataset. Built on scenes with annotations having parameters connected with triggers; this slideshow follows a template for visual consistency.

Elements of Narration

  1. Scenes

    2. The slides that make up the presentation. These slides can be navigated through guding arrows in the corner. Press Esc key at any point to get an overview of all scenes.

  2. Annotations

    3. Various tooltips, labels & legends; that display dish names and counts.

  3. Parameters

    4. The main parameters of this story are dish names, their counts and appearance years.

  4. Triggers

    5. Zoom and Brush filter, buttons with affordances are used to trigger updates in visuals based on new parameters.

Where now?

This was a very small attempt at using visuals to analyse the Menus dataset; we haven't yet visualised a very significant dimesnion for the dishes - the price; an obvious next step would be finding trends in dish prices - how prices have changed over the years?
Few important other parameters that came with data are - sponsor ( restaurants usually ), the occasion ( breakfast/ lunch etc ), venue (commercial, religious etc) and physical menu description of the menu itself - all of these can be visualised with help of D3 to convey more information about our culinary history.
